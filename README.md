Spring Boot CRUD API

Tecnologias
- Java
- Spring Boot
- PostgreSQL

Funcionalidades
- Cadastro
- Listagem
- Atualização
- Remoção

Objetivo
Projeto desenvolvido para praticar construção de APIs REST utilizando Spring Boot.

Endpoints
## Endpoints

| Método | Endpoint | Descrição |
|--------|----------|------------|
| GET | /usuarios | Lista usuários |
| POST | /usuarios | Cria usuário |
| PUT | /usuarios/{id} | Atualiza usuário |
| DELETE | /usuarios/{id} | Remove usuário |

Arquitetura
O projeto segue arquitetura em camadas:

- Controller
- Model
- Service
- Repository
